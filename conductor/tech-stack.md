# Tech Stack - Wellness Tracker

## Core Technologies
- **Language:** TypeScript (Strict Mode)
- **Frontend & Backend Framework:** React Router v7 (v7.10.1)
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4

## Data Management
- **Database:** MongoDB
- **Driver:** `mongodb` Node.js driver (v7.0.0)

## Tooling & Infrastructure
- **Build Tool:** Vite (via React Router)
- **Runtime:** Node.js
- **Package Manager:** npm

## Quality Assurance
- **Testing Framework:** Vitest
- **Testing Libraries:** React Testing Library, JSDOM
- **Linting:** ESLint (with `typescript-eslint` and `eslint-plugin-react`)
- **Type Checking:** TypeScript (tsc)
