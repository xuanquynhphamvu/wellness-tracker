# Product Guide - Wellness Tracker

## Product Vision
Wellness Tracker is a production-ready mental health quiz application designed to help individuals monitor their mental well-being through structured assessments and data-driven insights. It serves as both a functional health tool and a reference implementation for modern web patterns using React Router v7 and MongoDB.

## Primary Goals
- **Quiz Management:** Empower administrators to create and refine sophisticated mental health assessments using a dynamic quiz builder.
- **Progressive Insights:** Provide users with clear, actionable feedback on their wellness journey through intuitive data visualizations.
- **Modern Web Standards:** Maintain a high-performance, type-safe, and server-rendered codebase that demonstrates best practices in the React Router ecosystem.

## Target Users
- **Wellness Seekers:** Individuals looking for a private and reliable way to track their mental health trends over time.

## Core Features
- **Dynamic Quiz Builder:**
    - Custom score range mapping and interpretations.
    - Dynamic question management.
- **Progress Tracking:**
    - Interactive line charts for visualizing score trends over time.
    - Historical data overview.
- **Core Experience:**
    - Fast initial page loads via Server-Side Rendering (SSR).
    - Mobile-first, responsive interface.

## Non-Functional Requirements
- **Performance:** Optimized for minimal loading states and snappy interactions.
- **Responsive Design:** A mobile-first approach ensuring the application is accessible and functional on all device sizes.
- **Type Safety:** End-to-end TypeScript integration to ensure data integrity and developer productivity.
